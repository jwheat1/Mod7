Space Station Mystery Game

A text-based adventure game in Python.

Explore an abandoned space station orbiting Mars as a young space engineer. Solve puzzles, find clues, and uncover the mystery of the missing crew. Your choices affect the ending.

How to Play
Make sure you have Python 3 installed.
Download or clone the repository.

Open a terminal and navigate to the game folder:
cd "JW Milestone 3"

Run the game:
python3 JWstartgame.py

Follow the instructions in the game and type numbers or text to make choices.

Game Structure
JWstartgame.py – Main game loop and chapter selection
JWglobalvariables.py – Tracks player, inventory, and game progress
JWchapter1.py → JWchapter5.py – Each chapter’s story and player actions

Features:
Explore areas like Docking Bay, Laboratory, Greenhouse, and Control Room
Collect clues and items to progress

Multiple endings-
Good Ending: Send report to Earth
Secret Ending: Delete all data
Neutral Ending: Shut down the station

Beginner-friendly Python code using functions, loops, and conditionals
